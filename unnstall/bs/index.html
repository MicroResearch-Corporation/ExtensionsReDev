Can you make the sidebar permanently visible?

You can open the Bookmark Sidebar in the side panel which is provided by your browser. This side panel opens alongside the main content of a webpage and will stay there until you manually close it.

The sidebar is not working

As a privacy policy, the browser prohibits extensions from working on the default New Tab page. Bookmark Sidebar provides a similar but alternate New Tab page, replacing the default, where the sidebar can be used.

You can also use the sidebar in the side panel provided by your browser.

The sidebar is often getting in the way

You can configure the sidebar to not open at the full edge of your browser window, but, for example, only when moving the cursor at the lower edge of the window. You might want to give that a try.

You can also use the sidebar in the side panel provided by your browser. This side panel opens alongside the main content of a webpage.

How can I rearrange my bookmarks?

Important - You can only rearrange entries when the display order of the list is set to "User-defined". To change this, click on the according icon in the header at the top of the sidebar.

You can move any entry in the sidebar by using drag & drop. To move an entry, hover the mouse pointer over the entry to be moved. The "Move entry" icon will then appear to the right of the entry. Move the mouse pointer over the "Move entry" icon and the pointer will change to a hand style pointer. Click and HOLD the left mouse button on the "Move entry" icon and drag the entry up or down into its new position. Release the left mouse button when the entry is in its new position.

To cancel, simply move the cursor outside the sidebar and release the mouse button.

How can I change the search engine on the new tab page?

If you open the new tab page, you can see an edit icon in the lower corner of the page. If you click on it, you will enter the edit mode. There, you can edit the search engine by clicking the icon in the search field.

In the overlay you can select one of the popular search engines or a custom one by providing the search results URL and the homepage URL. After setting everything up, don't forget to press "Save changes" and leave the edit mode, so your changes will persist.

How can I change the font size and line height?

You can change the line spacing and the font size of the entries in the sidebar, besides many other things. You might want to give that a try.

How can I open the sidebar with a keyboard shortcut?

You can set your keyboard shortcut for this extension.

I don 't want to click the search icon to reveal the search field

You don't have to click the search icon to reveal the search field. Once you open the sidebar, you can start typing right away to start a search. The search field will then be displayed automatically.

How can I blacklist the sidebar on some websites?

You can configure url filters to change where the sidebar should be displayed and where it shouldn't be loaded. For example, If you have a website, you regularly work with, and which conflicts with the sidebar, you can add a rule to not open the sidebar on this page.

If you need help defining the url rules for your black- or whitelist, just send me a message, so I can help you with this.

When I zoom a website, the sidebar is getting larger, too

When the sidebar is configured to open on top of other websites, it will be affected by the zoom level of that website. Ignoring the user zoom for certain objects of a page is considered counter intuitive and therefore Google removed the possibility of preventing some objects from being zoomed. The sidebar is affected by this restriction as well and unfortunately there is no way of bypassing this anyhow.

The sidebar in the side panel provided by the browser is not affected by the page zoom level.
